# Otacon Changelog

All notable changes to Otacon will be documented in this file.

## [1.0.0] - 2026-01-30

### Added
- Initial release of Otacon
- One-click installer for Windows and macOS
- Simplified wrapper around OpenClaw
- Docker Compose setup
- Simple CLI commands (otacon-start, otacon-stop, otacon-help)
- Auto-configuration with sensible defaults
- WebChat interface (ported from OpenClaw)
- Desktop shortcuts creation
- Auto-start on login

### Features
- Pre-configured with GPT-4o-mini as default model
- Secure password generation
- Automatic Node.js installation (if needed)
- Windows: NSIS installer builder
- macOS: DMG installer builder
- Cross-platform portable binaries
